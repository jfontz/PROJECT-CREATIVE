# Animated Project Cards

This project creates a webpage with animated project cards on hover showcase.

## Description

It includes four project cards with project information on the left and project images on the right. 
Upon hovering over a card, the left side expands, revealing more details (prompt), and the right side fades out.
A share icon also appears smoothly in the top-right corner when hovering over a card. The projects are arranged in a slightly staggered manner upon hover.

## Technologies Used

- HTML
- CSS
